IPv4

Since the dawn of the Internet, IPv4 has been used as the standard in addressing protocol. This protocol worked by using 32 bits of data for a total of 4.3 billion unique addresses. Now with the exponential increase in devices going onto the internet, 32 bits is no longer enough. In comes IPv6 ("IPV6 Addressing").
